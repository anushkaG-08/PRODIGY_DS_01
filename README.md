# PRODIGY_DS_01

## Task 01 – World Population Data Analysis

Performed data cleaning and exploratory analysis on a global population dataset.
Focused on understanding population trends across countries and extracting meaningful insights from structured data.

## Work Done:

Loaded and inspected large CSV dataset using Pandas

Checked data types, summary statistics, duplicates, and missing values

Removed unnecessary columns for better analysis

Identified top 10 most populated countries

Compared population data for different years (e.g., 2016 vs 2022)

Visualized results using bar plots

## Skills Used:
Data Cleaning • EDA • Pandas • Data Filtering • Data Visualization
